---
title: Permissions
updated: July 19, 2026
---

| Permission | Default | Description |
|------------|---------|-------------|
| `valentines.use` | Everyone | Use the main interface and commands |
| `valentines.effect` | Everyone | Use player effects |
| `valentines.marry` | Everyone | Use the marriage feature |
| `valentines.divorce` | Everyone | End a marriage |
| `valentines.admin` | Operators | Use administrative features |
| `valentines.reload` | Operators | Reload the plugin configuration |
| `valentines.friends.use` | Everyone | Use the friends interface |
| `valentines.friends.add` | Everyone | Send friend requests |
| `valentines.friends.*` | Operators | Grant every friends permission |
